Explore Sweden's Ancient Runestones

An interactive map application allowing you to discover and explore 6,815+ Swedish runestones with detailed information and offline capabilities.

Open Application

Features

Interactive Map

Clustering map with 6,815+ runestones across Sweden. Easily find stones near you or search for specific locations.

Offline Access

Offline caching of runestones ensures you can explore even in remote areas without internet connection.

Rich Information

Detailed data for each runestone, including name, location, and historical attributes.

Gamified Experience

Create an account to track your visited runestones and build your profile as an explorer.

Modern Tech

Built with React 19, MapLibre GL, and Supabase for a fast, real-time experience.

Open Data

Leveraging data from OpenFreeMap and Samnordisk Runtextdatabas for accurate historical information.

Data Sources

Map Tiles

OpenFreeMap provides the beautiful and fast map tiles used in our interactive explorer.

Runestone Data

Samnordisk Runtextdatabas is our primary source for accurate runestone inscriptions and historical data.

Database

Leveraging the original SQLite database from Rundata-net for comprehensive record management.
